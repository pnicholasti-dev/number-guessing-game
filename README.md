(PT/BR)# 🎯 Number Guessing Game (Java)

Este é um jogo simples desenvolvido em Java onde o jogador deve adivinhar um número aleatório entre 1 e 100.

## 📌 Funcionalidades

- Geração de número aleatório entre 1 e 100
- Contagem de tentativas
- Dicas ao jogador:
  - "Too low" (muito baixo)
  - "Too high" (muito alto)
- Loop até o usuário acertar o número

## ⚙️ Como funciona

1. O programa gera um número aleatório de 1 a 100  
2. O usuário tenta adivinhar o número  
3. O sistema informa se o chute está alto ou baixo  
4. O jogo termina quando o número correto é encontrado  

## 🧠 Lógica do jogo

- Uso da classe `Random` para gerar números aleatórios
- Uso de `do-while` para repetição até acertar
- Contador de tentativas
- Estrutura condicional (`if / else if / else`)

## 🛠️ Tecnologias utilizadas

- Java
- Random
- Scanner
- Estruturas de repetição

  (EN)# 🎯 Java Number Guessing Game

This is a console-based number guessing game developed in Java to practice loops, conditionals, and random number generation.

## 🚀 Objective

Create an interactive game where the user must guess a randomly generated number between 1 and 100.

## ⚙️ Features

- Random number generation (1–100)
- User input handling
- Feedback system:
  - Too low
  - Too high
- Attempt counter
- Game loop until correct answer

## 🧠 Concepts practiced

- `Random` class usage
- `Scanner` input handling
- `do-while` loop
- Conditional statements
- Basic game logic

## 📌 Result

The game continues until the user guesses the correct number, showing the number of attempts required to win.

Project developed for Java practice and logic improvement.
